# Important elements inside `<head>`

## `<title>` element

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Website title</title>
  </head>
</html>
```

## `<meta>` element

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="author" content="Parth Maru" />
    <meta
      name="description"
      content="This page contains information about HEAD element"
    />
  </head>
</html>
```

## `<link>` element

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="author" content="Parth Maru" />
    <meta
      name="description"
      content="This page contains information about HEAD element"
    />

    <!-- Adding a fav icon -->
    <link rel="icon" href="fav-icon.png" type="image/x-icon" />

    <!-- Adding stylesheet -->
    <link rel="stylesheet" href="style.css" type="text/css" />
  </head>
</html>
```
