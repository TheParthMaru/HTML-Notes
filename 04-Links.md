## Anchor element

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Links in HTML</title>
  </head>

  <body>
    <p>Click here to go to <a href="https://www.google.com">Google</a>.</p>
    <p><a href="about.html">About me</a></p>
    <p><a href="#important">Go to important text</a></p>

    <p>Important text below:</p>
    <p id="important">Lorem ipsrumfas fabkfbafb afafnkenf</p>

    <a href="html5.png" download>HTML5 Icon</a>
    <a href="mailto:random@email.com">Contact me</a>
    <a href="https://www.google.com" target="_blank">Google in new tab</a>
    <a href="/">Home/ Default page</a>
  </body>
</html>
```
