# Text Basics

## Headings

- Starts from <h1> to <h6>.

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <h1>Largest heading</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
    <h5>Heading 5</h5>
    <h6>Smallest heading</h6>
  </body>
</html>
```

## Paragraph

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p>This is a paragraph</p>
    <p>This is another paragraph</p>
  </body>
</html>
```

## Horizontal rule

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <hr />
    <p>This is a paragraph</p>
    <hr />
  </body>
</html>
```

- In HTML, whitespaces are collapsed if extra whitespaces are inserted.

## Line break

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p>
      Let me tell you how: <br />...I learned more about Web Dev. <br />...I
      plan out my schedule. <br />...I use resources from MDN.
    </p>
  </body>
</html>
```

## Emphasize text

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p>I am looking for a <em>Frontend Web Developer</em> profile.</p>
  </body>
</html>
```

## Strong element

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p>I am looking for a <strong>Frontend Web Developer</strong> profile.</p>
  </body>
</html>
```

## HTML Entities

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p>&nbsp; -> No breaking space</p>
    <p>&lt; -> Less than symbol</p>
    <p>&gt; -> Greater than symbol</p>
    <p>&copy; -> Copyright symbol</p>
  </body>
</html>
```

- More entities -> https://html.spec.whatwg.org/multipage/named-characters.html#named-character-references

## Abbrevation

```html
<!DOCTYPE html>
<html lang="en">
  <body>
    <p><abbr title="Mozilla Developer Network">MDN</abbr></p>
  </body>
</html>
```
